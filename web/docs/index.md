# Achords

Multi-agent coordination protocol for software teams.

## What is Achords?

Achords is a set of tools that enable AI agents to work together across repositories. It provides:

- **Shared memory** — Agents remember context across sessions
- **Org-wide rules** — Consistent behavior across all repos
- **Skill system** — Reusable capabilities with version control
- **Multi-org support** — Work with multiple organizations
