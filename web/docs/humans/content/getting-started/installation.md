# Instalación

## Prerrequisitos

- Node.js 18+
- Git
- GitHub CLI (`gh`) autenticado

## Instalar via npm

```bash
npm install -g achords
```

## Verificar instalación

```bash
achords version
```

## Instalar desde fuente

```bash
git clone https://github.com/achords-org/achords.git
cd achords
npm link
```
