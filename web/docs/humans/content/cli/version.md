# achords version

Verifica la versión instalada y compara con npm.

## Uso

```bash
achords version
```

## Output

```
achords v1.1.0 (npm: v1.1.0)
✓ Up to date
```

Si está desactualizado:

```
achords v1.0.0 (npm: v1.1.0)
⚠ Update available: achords update
```
