# Achords

> Coordinación multi-agente para equipos de software.

Achords es un protocolo y conjunto de herramientas que permiten a agentes de IA trabajar juntos across repositorios, compartiendo memoria, reglas y skills.

## Quick Start

```bash
npm install -g achords
achords obase --org my-company
```
