# Installation

## Prerequisites

- Node.js 18+
- Git
- GitHub CLI (`gh`) authenticated

## Install via npm

```bash
npm install -g achords
```

## Verify installation

```bash
achords version
```

## Install from source

```bash
git clone https://github.com/achords-org/achords.git
cd achords
npm link
```
