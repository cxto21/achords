# achords version

Check installed version and compare with npm.

## Usage

```bash
achords version
```

## Output

```
achords v1.1.0 (npm: v1.1.0)
✓ Up to date
```

Or if outdated:

```
achords v1.0.0 (npm: v1.1.0)
⚠ Update available: achords update
```
